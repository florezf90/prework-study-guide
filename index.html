CSS

  • There are 3 ways to style with CSS, Internal, external and inline CSS
  • Inline CSS: apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal CSS: it's when we use a "style" element in the "head" section of the HTML file
  • External CSS: it's when we create a separate file to write our CSS code so it doesn't get mixed up with the HTML code
  • when we want to link an external css file to our HTML file, use the following command: "h1 style="color:red;">Prework Study Guide"/h1"
